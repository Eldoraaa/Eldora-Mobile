# Mobile Eldora

Expo React Native app for Eldora caregivers.

## Stack
- Expo + React Native
- TypeScript
- Expo Router
- TanStack Query
- Zustand + AsyncStorage
- NativeWind/Tailwind-style classes

## Main features
- Multi-home selection
- DoraBot WiFi setup and local pairing
- Device list, rooms, and device management
- Scene automation and tap-to-run scenes
- Alerts and notification center
- Voice configuration per device

## Run
```bash
npm install
npm run start
```

## Check
```bash
npx tsc --noEmit
```
